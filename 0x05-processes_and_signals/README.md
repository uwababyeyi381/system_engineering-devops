# 0x05. Processes and signals

This directory contains my Bash scripts for learning how to manage Linux processes and handle signals. 

In this project, I learned:
* What a Process ID (PID) is and how to find it.
* How to list running processes using `ps` and filter them with `pgrep`.
* How to gracefully stop processes using `kill` and `pkill` (SIGTERM).
* How to forcefully terminate unresponsive processes (SIGKILL).
* How to write infinite `while` loops and pause them with `sleep`.
* How to use the `trap` command to catch and respond to system signals.
